Sobre o Makerspace

Um Makerspace é um lugar onde as pessoas se reúnem para compartilhar recursos e conhecimento, trabalhar em projetos, inventar e construir utilizando uma variedade de ferramentas e materiais, tornando-se um ambiente para desenvolver a criatividade, a inovação e a mentalidade de colaboração em comunidade. Baseado neste pensamento, o ARARA Makerspace foi criado por alunos do curso de graduação em Engenharia de Computação do campus UFSC Araranguá, com objetivo disseminar esta cultura a partir de um laboratório aberto a comunidade de Araranguá.

Um espaço para construir suas idéias.

Este movimento é baseado na cultura DIY (Do It Yourself, ou, faça você mesmo) onde influencia as pessoas a tentarem construir ou montar um produto ou dispositivo, ao invés de comprá-lo. Para isso, temos um laboratório comunitário com equipamentos para desenvolver diversos projetos.

Compartilhamento de conhecimento.

Mais que o espaço e os materiais, os Makerspaces são ambientes de aprendizado colaborativo onde as pessoas se reúnem para compartilhar conhecimento e aprender novas habilidades. Com esse intuito, realizamos diversos cursos e workshops com foco em programação e eletrônica para a comunidade de Araranguá.

Cursos

Data Analytics em Python

O curso abrangeu uma visão geral sobre fundamentos de Ciência de Dados, utilizando ferramentas atuais de análise e visualização de dados em python.

Curso de Arduino

Neste curso foi ensinado tópicos básicos de programação e eletrônica utilizando Arduino para o desenvolvimento de projetos.

Curso de Lógica de programação

Este curso foi ministrado para crianças com o intuito de desenvolver o raciocínio lógico, a partir de brincadeiras e jogos. No final, a crianças criam um jogo, utilizando a plataforma Code.org.
